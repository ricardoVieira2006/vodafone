# Organizador de Propostas/Vendas Vodafone Fibra

## Como usar:

1. Abra o arquivo `index.html` no seu browser (duplo-clique ou arraste para Chrome/Firefox).
2. Os dados são guardados automaticamente no browser (localStorage).
3. **Adicionar**: Clique "+ Adicionar Nova Proposta", preencha os campos, carregue fotos, guarde.
4. **Editar/Eliminar**: Na tabela, use os ícones ✏️ ou 🗑️.
5. **Buscar**: Digite nome do cliente no campo de busca.
6. **Fotos**: Clique nas miniaturas para ver em tamanho grande.

## Funcionalidades:
- Campos: Nome cliente, Pacote/descrição, Preço, IBAN, Foto Cartão Cidadão, Fatura NOS.
- Responsivo (funciona no telemóvel).
- Pesquisa por nome.
- Persistência local (não perde dados ao fechar).
- Previsão de fotos (base64).

Para backup: Abra DevTools (F12) > Application > Local Storage > copie o valor de `vodafone_sales_data`.

Feito com HTML/CSS/JS puro - sem instalação necessária!
